## 测试流程

### 步骤1: 环境准备（1分钟）

```bash
# 检查Python版本
python --version  # 期望: Python 3.8+

# 检查依赖包
pip list | grep -E "numpy|pandas"

# 安装缺失依赖（如需要）
pip install numpy pandas
```

### 步骤2: 运行健康检查（2分钟）

```bash
python scripts/health_check.py
```

**如果失败**:
- 检查模块导入错误
- 确认config.yaml存在
- 查看详细错误信息

### 步骤3: 运行功能测试（5分钟）

```bash
# 快速回测
python scripts/quick_backtest.py

# 异常检测
python scripts/test_anomaly.py
```

**如果失败**:
- 检查数据生成
- 查看日志文件: `logs/trading_system.log`

### 步骤4: 运行集成测试（2分钟）

```bash
./start.sh
```

**如果失败**:
- 逐项检查每个步骤
- 确认所有测试通过

## 故障排除

### 问题1: 健康检查失败

**症状**: 健康得分 < 80

**解决方案**:
```bash
# 检查模块导入
python -c "from scripts.event_bus import get_event_bus; print('OK')"

# 检查配置文件
ls -la config.yaml config.json

# 检查数据目录
ls -la assets/data
```

### 问题2: 回测无交易

**症状**: 交易次数 = 0

**解决方案**:
- 检查数据长度是否足够
- 增加测试数据样本数
- 检查策略参数

### 问题3: 异常检测误报

**症状**: 正常数据被检测为异常

**解决方案**:
- 调整阈值配置
- 检查数据格式
- 查看异常检测日志
