## 2. 策略局限性

### 2.1 结构性挑战

1. **高趋势市场占比**: 2025-2026年，加密市场99%时间处于强趋势(ADX>30)
2. **盈亏平衡胜率偏高**: ATR=3.0×止损 → 盈亏平衡需要 1/(1+3×0.74) = 31%... 实际上胜率需 >53.1%
3. **止损幅度大**: ATR=3.0× ≈ 1.5-2.0%止损，单笔亏损较大
4. **BTC适应性差**: BTC均值回归窗口短，反弹容易被趋势吃掉

### 2.2 为什么 Gap 难以大幅突破

- 胜率 52-58% 接近盈亏平衡 (53-54%)
- 盈亏比 0.74-0.89 < 1.0 → 止盈不如止损大
- Gap = WR - 1/(1+RR): 当 RR<1 时，需要 WR>50% 才能正期望
- 最优情况: WR=58%, RR=0.74 → Gap=2.19%

## 3. 优化方向建议

### 3.1 短期可实施的改进

1. **分离ETH/BTC参数**: ETH用更宽止损(ATR=3.0)，BTC用更紧止损(ATR=2.0)
2. **动态止盈**: TP_mult从固定的2.0改为ADX自适应
3. **品种选择**: 优先ETH，放弃BTC或降低BTC权重
4. **非对称止损**: 做多止损 ATR=2.5，做空禁用

### 3.2 中期研究方向

1. **多周期确认**: 用4H确认趋势方向，用1H寻找入场点
2. **流动性过滤**: 仅在成交量放大时入场
3. **协整套利**: ETH/BTC配对交易，对冲市场方向风险
4. **外部数据**: 引入链上数据、期权隐含波动率作为市场状态判断

### 3.3 策略改进方向

1. **趋势跟踪补充**: 在强趋势市场(ADX>40)用趋势跟踪替代均值回归
2. **机器学习排序**: 用随机森林/梯度提升选择最优品种/方向
3. **Walk-Forward分析**: 更严格的样本外验证
