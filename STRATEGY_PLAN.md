# 极简订单流超短线策略 v1.0 — 完整方案

## 一、策略核心

### 核心理念
**不做方向预测，只跟随资金流**。放弃所有滞后指标（RSI/MACD/布林带），只用 3 个原始指标：

| 优先级 | 指标 | 逻辑 | 有效性 |
|--------|------|------|--------|
| 1 | 主动成交量净额 | taker_buy - taker_sell = 大资金方向 | ⭐⭐⭐⭐⭐ |
| 2 | 价格突破 | 突破 N 周期高低点 = 动能爆发 | ⭐⭐⭐⭐ |
| 3 | 成交量突增 | vol > M × MA20 = 资金入场确认 | ⭐⭐⭐ |

### 入场条件（同时满足）
1. **主动成交量**：净主动量 > 阈值 × 过去 20 周期均值
2. **价格突破**：做多=突破 20 周期高点，做空=突破 20 周期低点
3. **成交量放大**：当前量 > 2.5-3.0 倍均量（增强信号）

### 出场条件（任一满足）
- 止盈：0.6-1.0%（按品种波动率）
- 止损：0.5-0.8%（紧止损）
- 时间止损：3-5 根 K 线无动能即出

## 四、风控规则（硬性）

### 单笔风险
```
- 单笔最大亏损：≤2% 总资金
- 例：1000U 账户 → 单笔最多亏 20U
- 根据止损%反推仓位：仓位 = 20U / 止损%
```

### 日内风控
```
- 日最大亏损：5% 总资金（触及即停）
- 同时持仓：最多 2 品种
- 单品种最多 1 仓
```

### 交易时段
```
- 最佳：20:00-24:00 UTC（美盘，波动最大）
- 避免：重大数据发布前后 15 分钟（CPI/NFP/美联储）
- 低波动时段暂停：ATR(14)/close < 0.15%
```

## 五、资金管理

### 起始资金建议
```
- 最小：500U（可开 0.01 BTC 或 0.1 ETH）
- 推荐：1000-2000U（分散 2-3 品种）
- 杠杆：1-3 倍（绝不超 5 倍）
```

### 仓位计算
```python
# 例：1000U 账户，ETH 当前价 3000，止损 0.7%
risk_amount = 1000 * 0.02 = 20U  # 单笔风险 2%
stop_loss_pct = 0.007
position_size = risk_amount / stop_loss_pct = 2857U  # 名义仓位
quantity = 2857 / 3000 = 0.95 ETH  # 实际开仓量
leverage = 2857 / 1000 = 2.85x  # 实际杠杆
```

## 八、风险提示

⚠️ **必须知晓的风险**

1. **样本量不足** — 当前仅 16 小时数据，需 7-30 天验证
2. **过拟合风险** — 参数可能仅适应当前市场
3. **流动性风险** — 极端行情滑点可能超 0.5%
4. **技术风险** — WebSocket 断线、API 限流、延迟
5. **手续费侵蚀** — 高频交易手续费是最大成本

### 应对策略
```
- 至少 100 笔交易样本再实盘
- 模拟盘运行 1 周无问题再上真金
- 设置 API 断线自动停止交易
- 每周复盘调整参数
```
